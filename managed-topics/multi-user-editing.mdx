Use live collaboration to allow multiple team members to edit Managed Topics at the same time without overwriting each other's work. Changes sync in real-time across all connected editors.

## How it works

When you open a Managed Topic for editing, your session is registered in Agent Studio. As you and your teammates make changes, the system tracks edits and synchronizes them across all active sessions:

- **Real-time updates**: Changes appear instantly in other editors' views
- **Conflict prevention**: The system prevents simultaneous edits to the same field by locking that field for other users while one person is actively editing
- **Change history**: All edits are tracked and can be reviewed in version history if needed

## Best practices

- **Communicate with your team** about which topics you're working on to avoid working on the same topic simultaneously
- **Save frequently** to ensure your changes are persisted
- **Use descriptive topic names** so teammates can quickly identify which topics are being edited
- **Keep editing sessions brief** when possible to minimize lock time for other collaborators

## Troubleshooting

**My changes aren't syncing** — Refresh the page to reload the latest version. If the issue persists, check your internet connection.

**I can't edit a field** — Another teammate may be actively editing that field. Wait for them to finish or ask them to complete their edits.
