## Overview

NICE CXOne is a cloud-based contact center platform that provides omnichannel customer engagement capabilities. The PolyAI integration enables seamless handoff between AI agents and human agents within the CXOne environment.

## Capabilities

- **Signal handling**: Send signals to CXOne for call routing and agent assignment
- **Call handoff**: Transfer conversations from AI to human agents
- **Context preservation**: Pass conversation context and metadata to CXOne agents

## Configuration

To integrate PolyAI with NICE CXOne, you'll need to configure the following parameters:

### Required credentials

- **API endpoint**: Your CXOne API endpoint URL
- **Authentication credentials**: API key or OAuth credentials for CXOne access
- **Tenant ID**: Your CXOne tenant identifier

### Integration setup

Contact your PolyAI representative to configure the CXOne integration with your specific endpoint and authentication details.

## Usage

Once configured, your PolyAI agent can trigger handoffs to CXOne agents using the appropriate signal actions in your conversation flows.
