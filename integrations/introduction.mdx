Connect PolyAI to telephony, CRM, payments, and knowledge platforms. Integrations let your agent access external systems during calls — hand off to live agents, look up reservations, create support tickets, or retrieve knowledge base articles.

All integrations listed here are available from **Configure > Integrations** in Agent Studio.

## Don't see your platform?

If your system isn't listed above, you can build a custom integration using the [APIs tab](/api/introduction). Define HTTP endpoints, configure per-environment base URLs and authentication, and call them from Python functions using `conv.api` — no pre-built connector needed.

For integrations not available in the Studio UI (Custom SIP, PCI Pal, Stripe, and others), see the [Managed services](/integrations/managed-services) section.

</div>
