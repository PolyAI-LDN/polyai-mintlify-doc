title: 'Access control'
description: 'Manage which agents can access secrets'
---

Restrict which agents can access sensitive information in the Secrets Vault.

## Managing access

1. Open the secret details page
2. Scroll to **Agent access**
3. Select which agents should have access

## Principle of least privilege

Grant access only to agents that require specific secrets for their functions. This minimizes the risk of unauthorized access.
