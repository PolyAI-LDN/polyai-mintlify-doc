# Secrets

The Secrets Vault securely stores API keys, tokens, credentials, and other sensitive information. Use it to manage secrets, integrate with external systems, and enhance your agents' capabilities.

## Importance

The Secrets Vault enables you to securely store sensitive information with per-secret access control. This provides an additional layer of security over your data and allows you to create powerful function calls that integrate with external APIs.
