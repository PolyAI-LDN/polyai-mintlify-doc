<Info>
Functions run on **Python 3.14**. Only the [standard library](https://docs.python.org/3.14/library/index.html) and the packages listed below are available. You cannot install additional packages.
</Info>

## Standard library

Within functions you have access to all of the [Python 3.14 standard library](https://docs.python.org/3.14/library/index.html), including modules like `datetime`, `json`, `re`, `urllib`, `hashlib`, and more.

## Notable Python 3.14 features

If you're coming from Python 3.9, the following features are now available in your functions:

- **Structural pattern matching** (`match`/`case`) — a cleaner alternative to long `if`/`elif` chains for branching on values or structures. See the [Python docs on match statements](https://docs.python.org/3.14/tutorial/controlflow.html#match-statements).
- **Parenthesized context managers** — you can now use parentheses to split multiple context managers across lines:
  ```python
  with (
      open("input.json") as infile,
      open("output.json", "w") as outfile,
  ):
      ...
  ```
- **Exception Groups and `except*`** — handle multiple exceptions raised concurrently. Useful when your function makes parallel API calls. See [PEP 654](https://peps.python.org/pep-0654/).
