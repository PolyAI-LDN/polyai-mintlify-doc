<Info>
Functions run on **Python 3.14**. Only the [standard library](https://docs.python.org/3.14/library/index.html) and the packages listed below are available. You cannot install additional packages.
</Info>

## Standard library

Within functions you have access to all of the [Python 3.14 standard library](https://docs.python.org/3.14/library/index.html), including modules like `datetime`, `json`, `re`, `urllib`, `hashlib`, and more.

## Non-standard libraries

We also include the following packages:

| Library | Version | Description |
|---------|---------|-------------|
| [requests](https://requests.readthedocs.io/en/latest/user/quickstart/) | 2.28.2 | HTTP library for making API requests |
| [decorator](https://pypi.org/project/decorator/4.4.2/) | 4.4.2 | Function decorator utilities |
| [urllib3](https://pypi.org/project/urllib3/1.26.18/) | 1.26.18 | Low-level HTTP client |
| [jsonschema](https://pypi.org/project/jsonschema/3.0.2/) | 3.0.2 | JSON schema validation |

<Tip>
The `requests` library is the recommended way to make external API calls from your functions. See the [requests quickstart guide](https://requests.readthedocs.io/en/latest/user/quickstart/) for usage examples.
</Tip>
