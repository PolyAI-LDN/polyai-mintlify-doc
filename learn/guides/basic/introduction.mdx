## Your learning journey

<Steps>
  <Step title="Foundation and orientation">
    1. What is Agent Studio? (the dual build-and-analyse platform concept)
    2. Platform architecture overview (LLM/Raven, Knowledge Base, Flows, Retrieval Engine, STT/TTS)
    3. Agent Design Principles (seven principles covering trust, guiding the caller, robustness, feedback, correction, efficiency, and natural speech)
    4. Platform UI Tour
  </Step>
  <Step title="Builder Tool Suite (Basics)">
    1. Agent Rules
    2. Knowledge (Managed Topics and Connected Knowledge)
    3. Local Chat (basic testing)
    4. Environment and Branching Basics
    5. Flows
    6. Tools
  </Step>
  <Step title="Operating a Live Agent">
    1. The Improvement Loop (data → insight → change → promote)
    1. Conversation Review Basics
    2. Smart Analyst Basics
  </Step>
  <Step title="Building Agents Theory">
    1. Building Agents Methodology
    2. Adding a new use case on an existing agent
  </Step>
  <Step title="Practical: Putting it all together">
    1. Building your first production Agent
    2. Adding a use case
  </Step>
  <Step title="Knowledge Check">
    - Time to check what you've learnt! (don't worry, this isn't an assessment)
  </Step>
</Steps>
