Welcome to **PolyAcademy** level one, where you can create a complete, working deployment you can test in both web **Chat** and
using voice **Call**. We will review the **Conversation review**, and how to safely promote a draft to a published agent.

Follow the completion checklist steps in order:

## Completion checklist

You are done with **level one** of the PolyAcademy when all of the following are complete:

- [Project is created using a Level 1 setup](/learn/guides/get-started/create-a-project)
- [Agent behaviour is defined (non-domain-specific)](/learn/guides/get-started/edit-agent-behavior)
- [A simple FAQ KB topic works in Chat](/learn/guides/get-started/add-kb-topic)
- [An SMS with a handoff knowledge base topic works using built-ins](/learn/guides/get-started/add-complex-kb-topic)
- [Response control is linked to a function](/learn/guides/get-started/response-control)
- [A voice is selected and validated in Call](/learn/guides/get-started/edit-voice-settings)
- [The audio cache is understood and managed](/learn/guides/get-started/audio-management)
- [Any global ASR biasing or corrections are configured](/learn/guides/get-started/global-asr)
- [Changes are tested and versioned in environments](/learn/guides/get-started/environments)
- [Agent behaviour is viewable in Conversation review](/learn/guides/get-started/conv-review)
- [Variants are configured, and tested](/learn/get-started/guides/variants)
