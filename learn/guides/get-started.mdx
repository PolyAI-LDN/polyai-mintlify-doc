## Your learning journey

<Steps>
  <Step title="Create your project">
    Set up a new Agent Studio project with proper configuration
  </Step>
  <Step title="Customize agent behavior">
    Define personality, rules, and baseline behavior
  </Step>
  <Step title="Add simple topics">
    Build and test FAQ-style Managed Topics
  </Step>
  <Step title="Configure voice">
    Select and tune your agent's voice settings
  </Step>
  <Step title="Master environments">
    Learn to test and promote changes safely
  </Step>
  <Step title="Review conversations">
    Understand what happened and why
  </Step>
</Steps>
