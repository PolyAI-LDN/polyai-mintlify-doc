## How it works

Each lesson follows an interactive structure designed to reinforce learning as you go:

<Steps>
  <Step title="Learn">
    Read through the concept with examples and best practices
  </Step>
  <Step title="Quiz">
    Test your understanding with interactive quizzes placed throughout each lesson
  </Step>
  <Step title="Try it yourself">
    Apply what you learned with a hands-on challenge
  </Step>
  <Step title="Track your progress">
    Mark lessons complete and see how far you've come
  </Step>
</Steps>

<Note>
  Each lesson includes **3 interactive quizzes** distributed throughout the content. Level 1 quizzes focus on recall, while Level 2 quizzes present real-world scenarios to diagnose.
</Note>

## What you'll learn

<AccordionGroup>
  <Accordion title="Level 1: Fundamentals" icon="circle-1">
    - Create and configure your first agent
    - Build simple FAQ topics
    - Test in Chat and Call
    - Manage environments and versions
    - Review and diagnose conversations

  <Accordion title="Level 2: Advanced techniques" icon="circle-2">
    - Design complex multi-turn topics
    - Understand functions and return values
    - Configure response controls

    - Master variant management
    - Build and design flows for structured interactions
  </Accordion>

  <Accordion title="Level 3: Expert" icon="circle-3">

    - Write natural-sounding agent speech
