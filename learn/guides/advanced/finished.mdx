<AccordionGroup>
  <Accordion title="Quick check: What do Response Controls regulate?">
    **Answer:** The agent's **own output** — not what the user says. They run after the LLM generates a response, before it's spoken or sent.
  </Accordion>
  <Accordion title="Quick check: When should you use Transcript Corrections?">
    **Answer:** When ASR **consistently mishears one thing as another** (e.g., "refund" → "reef fund"). Use Keyphrase Boosting when the word is correct but under-recognised.
  </Accordion>
  <Accordion title="Quick check: At Level 2, the transcript is considered...?">
    **Answer:** Only a **symptom**. The real diagnostic work happens in diagnosis layers, function traces, and variant attribution.
  </Accordion>
</AccordionGroup>
