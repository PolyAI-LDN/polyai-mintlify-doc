Edit, add, or bulk-update Managed Topics content. Test in Sandbox before promoting to Live.

## Quick reference

| I need to... | Action | Time estimate |
|--------------|--------|---------------|
| Edit a single answer | Click topic → Edit answer → Save | 2 min |
| Add a new topic | + Topic → Add questions & answer → Save | 5 min |
| Update 10-20 topics | Export CSV → Edit → Import CSV | 15 min |
| Update 50+ topics | Export CSV → Edit → Import CSV | 30 min |
| Fix wrong topic triggering | Add sample questions to correct topic | 3 min |
| Merge duplicate topics | Combine content → Delete duplicate | 5 min |
| Test changes | Agent Chat in Sandbox | 5 min |
| Promote to Live | Publish → Promote to Pre-release → Promote to Live | 5 min |

<Tip>
   Always test in _Sandbox_ first and promote only when you're happy. Details are in [Environments & versions](/environments-and-versions/introduction). 
</Tip>

## Edit an existing topic

<Steps>
  <Step title="Select the topic">
    Click the topic you want to change.
  </Step>
  <Step title="Update the answer">
    Edit text directly in the _Answer_ panel.
  </Step>
  <Step title="Save your changes">
    Click **Save**.
  </Step>
  <Step title="Test and publish">
    Use **Publish** and run tests in Sandbox before promoting.
  </Step>
</Steps>

<AccordionGroup>
  <Accordion title="Common fixes" icon="wrench">
    | Issue                            | What to do                                                      |
    | -------------------------------- | --------------------------------------------------------------- |
    | The agent gives a partial answer | Add clarifying details to the _Answer_ field.                   |
    | It triggers the wrong topic      | Add a few of the user's actual phrasings as _Sample questions_. |
    | It overlaps another topic        | Merge the content or re-title one topic to be more specific.    |
  </Accordion>
</AccordionGroup>

## Add a new topic

<Steps>
  <Step title="Create the topic">
    Click **+ Topic**.
  </Step>
  <Step title="Enter a clear title">
    Use a descriptive name that reflects the user's intent.
  </Step>
  <Step title="Add sample questions">
    Add at least three _sample questions_ — think of real user wording.
  </Step>
  <Step title="Write the answer">
    Write the _Answer_ in plain language.
  </Step>
  <Step title="Save and preview">
    Save and test the topic in Agent Chat.
  </Step>
</Steps>

## Bulk edits

Large updates are faster in a spreadsheet.

<Steps>
  <Step title="Export your topics">
    Click **Export CSV** to download a CSV file containing all of your Managed Topics.
  </Step>
  <Step title="Edit in your spreadsheet app">
    Edit or add rows. Keep column headers exactly as exported. Leave IDs intact for topics you're editing; new rows (without an ID) create new topics.
  </Step>
  <Step title="Save and upload">
    Save as CSV, then click **Import CSV** and upload the file.
  </Step>
  <Step title="Review and confirm">
    Review the diff, then click **Confirm import**.
  </Step>
</Steps>

## Promote your changes

<Steps>
  <Step title="Open environments">
    Go to **Deployments > Environments**.
  </Step>
  <Step title="Select your version">
    Select the new _Sandbox_ version.
  </Step>
  <Step title="Promote to Pre-release">
    Click **Promote to pre-release** and confirm.
  </Step>
  <Step title="Promote to Live">
    Click **Promote to Live** and confirm.
  </Step>
</Steps>

## Common workflows

<Tabs>
  <Tab title="Weekly update" icon="calendar-week">
    <Steps>
      <Step title="Review analytics for knowledge gaps">5 min</Step>
      <Step title="Add or update 3-5 topics">15 min</Step>
      <Step title="Test in Agent Chat">5 min</Step>
      <Step title="Publish to Sandbox">2 min</Step>
      <Step title="Promote to Pre-release for UAT">3 min</Step>
      <Step title="Promote to Live">3 min</Step>
    </Steps>

    **Total time:** ~30 minutes
  </Tab>
  <Tab title="Seasonal refresh" icon="snowflake">
    <Steps>
      <Step title="Export current topics to CSV">2 min</Step>
      <Step title="Update seasonal information (hours, policies, etc.)">20 min</Step>
      <Step title="Import updated CSV">5 min</Step>
      <Step title="Test seasonal queries in Agent Chat">10 min</Step>
      <Step title="Publish and promote">5 min</Step>
    </Steps>

    **Total time:** ~45 minutes
  </Tab>
  <Tab title="Fixing a gap" icon="magnifying-glass">
    <Steps>
      <Step title="Identify the gap from call logs or analytics">5 min</Step>
      <Step title="Create new topic or update existing">5 min</Step>
      <Step title="Add sample questions matching user phrasing">3 min</Step>
      <Step title="Test with actual user queries">5 min</Step>
      <Step title="Publish and promote">5 min</Step>
    </Steps>

    **Total time:** ~25 minutes
  </Tab>
</Tabs>
