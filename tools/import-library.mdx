<Info>
**This page requires Python familiarity.** PolyAI's function runtime uses **Python 3.12**. Only the [standard library](https://docs.python.org/3.12/library/index.html) and the packages listed below are available. You cannot install additional packages.
</Info>

## Standard library

Within functions you have access to all of the [Python 3.12 standard library](https://docs.python.org/3.12/library/index.html), including modules like `datetime`, `json`, `re`, `urllib`, `hashlib`, and more.

## Notable features available in the runtime

If you're coming from Python 3.9, the following features are available in your functions:

- **Structural pattern matching** (`match`/`case`, introduced in Python 3.10) — a cleaner alternative to long `if`/`elif` chains for branching on values or structures. See the [Python docs on match statements](https://docs.python.org/3.12/tutorial/controlflow.html#match-statements).
- **Parenthesized context managers** (Python 3.10+) — you can use parentheses to split multiple context managers across lines:
  ```python
  with (
      open("input.json") as infile,
      open("output.json", "w") as outfile,
  ):
      ...
  ```
- **Exception Groups and `except*`** (Python 3.11+) — handle multiple exceptions raised concurrently. Useful when your function makes parallel API calls. See [PEP 654](https://peps.python.org/pep-0654/).

## Non-standard libraries

The following packages are pre-installed:

| Library | Version | Description |
|---------|---------|-------------|
| [requests](https://requests.readthedocs.io/en/stable/user/quickstart/) | 2.32.5 | HTTP library for making API requests |
| [decorator](https://pypi.org/project/decorator/5.2.1/) | 5.2.1 | Function decorator utilities |
| [urllib3](https://pypi.org/project/urllib3/2.6.3/) | 2.6.3 | Low-level HTTP client |
| [jsonschema](https://pypi.org/project/jsonschema/4.25.1/) | 4.25.1 | JSON schema validation |

If you encounter unexpected behavior with a library, contact your PolyAI representative.

<Tip>
The `requests` library is the recommended way to make external API calls from your functions. See the [requests quickstart guide](https://requests.readthedocs.io/en/stable/user/quickstart/) for usage examples.
</Tip>
