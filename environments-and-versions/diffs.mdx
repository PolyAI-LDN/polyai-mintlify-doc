![diff-kb](/images/release-notes/2502/kb-diff.png)

Compare versions side-by-side to review additions, modifications, and deletions before promoting changes.

## Key features

- **Side-by-side comparison** across sandbox, pre-release, and live environments
- **Change attribution** showing who made each change
- **Content diff** for responses and sample questions

## How it works

1. **Select versions to compare:** Choose two versions of the project item from the dropdown menu.
2. **Review differences:** Changes are highlighted in a structured format:
   - **<Icon icon="plus" iconType="solid" color="#2D8C66" /> Additions** mean a new project item entry.
   - **<Icon icon="minus" iconType="solid" color="#D64F4F" /> Deletions** means a project item entry has been deleted.
   - **<Icon icon="rotate" iconType="solid" color="#3B73D1" /> Edits** is the symbol applied to any existing but altered project item entry.
3. **Assess impact:** Use the comparison view to evaluate changes before promoting updates to the next stage.
4. **Exit comparison mode:** Return to the project item editor or switch to another version for further review.
