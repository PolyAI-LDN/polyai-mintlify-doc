PolyAI's **Agentic Dialog Platform** is the platform that brands trust to handle their most complex conversations, now open to every builder across the enterprise. Agents handle real customer conversations – answering questions, making bookings, resolving issues, and handing off to humans. Most agents run on [Raven](/agent-settings/raven), PolyAI's proprietary LLM with sub-300ms latency across 24+ languages.

## Choose your surface

| | Agent Studio | ADK | APIs |
|---|---|---|---|
| **Who it's for** | Non-technical users, visual workflow | Technical teams, engineering workflows | Enterprises with internal tooling |
| **How you work** | Visual UI – point and click | CLI and Python – local dev with Git | REST endpoints – programmatic control |
| **Best for** | Building and tuning agents interactively | Bulk authoring, version control, CI/CD | Programmatic control, automation |
| **Code required** | Optional | Yes (Python) | Yes (any language) |

All three surfaces manage the same agents. An agent built in Studio can be updated through ADK or APIs, and vice versa.
