## What free trial mode does

Free trial mode is a streamlined view of the docs for self-serve Agent Studio users on the **60-day free trial**. It hides enterprise-only features, developer tools, and advanced configuration so you only see what's available to you in the self-serve plan.

<CardGroup cols={2}>
  <Card title="Self-serve focused" icon="sparkles">
    Hides the **Developer**, **API reference**, and **Advanced** tabs and any pages tagged as enterprise-only or code-required.
  </Card>
  <Card title="Cleaner sidebar" icon="list-tree">
    Collapses sidebar groups like Tools, Secrets, Variant management, Test Suite, and Custom telephony that are not part of the free trial.
  </Card>
  <Card title="Inline developer notes" icon="chevron-down">
    Where a page mixes UI and code, the developer details stay but are tucked behind an expandable accordion.
  </Card>
  <Card title="Persists across pages" icon="bookmark">
    Once you turn it on, free trial mode stays on across pages and sessions until you click **Exit free trial mode**.
  </Card>
</CardGroup>

## How to exit

You can leave free trial mode at any time in three ways:

- Click the **Free trial — exit** pill floating in the top-right corner of any docs page.
- Click any link to an enterprise or developer page (for example, the **API reference** tab). Free trial mode turns itself off and takes you there in full-docs view.
- Come back to this page and click **Exit free trial mode**.

## Who is this for?

<Tip>
  **Use free trial mode if you are...**
  - Signed up at [studio.us.poly.ai](https://studio.us.poly.ai) and exploring the self-serve plan.
  - Building your first agent through Studio Assistant and the Agent Studio UI.
  - Looking for knowledge, voice, Polyphone, and analytics docs without enterprise-only or code noise.
</Tip>

<Note>
  **Use full docs if you are...**
  - On an Enterprise plan and need access to telephony, SMS, handoffs, custom metrics, or dashboards.
  - Writing Python tools or secrets.
  - Integrating through the ADK or REST APIs.
  - Configuring advanced flows, variant management, or custom telephony.
</Note>
