The **November 2025** PolyAI Agent Studio release introduces the new Knowledge Source Hub, KB topic activation controls, an in-app integrations catalogue, multi-channel dashboards, Smart Analyst visual updates, and improved SMS configuration.

Expand the items for details:

<AccordionGroup>

<Accordion title="Source Hub" icon="book" id="knowledge-source-hub">

Upload and manage multiple knowledge sources to power your KB — files, URLs, and external integrations.

![source-hub](/images/release-notes/2511/source-hub.png)

**What's new:**
- Add **multiple external knowledge sources** to populate your knowledge base.
- Supported formats include PDFs, text files, Word docs, CSVs, JSON, URLs, and CRM/helpdesk integrations.
- Assign each source to specific **client environments** and **variants**.
- Group sources for easier navigation and maintenance.
- Re-upload or re-scrape sources to refresh content when upstream material changes.
- Per-chunk **citations** appear in Conversation Review, so you can see exactly which source text was used.

</Accordion>

<Accordion title="Integrations overview page" icon="puzzle-piece" id="integrations-overview-page">

Browse a catalogue of third-party tools that Agent Studio can integrate with.

![integrations](/images/release-notes/2511/integrations.png)

**What's new:**
- New **Integrations** page listing supported tools across Telephony, Chat, CRM, Vertical, and Knowledge Base categories.
- Search by integration name or description to find what you need quickly.
- Filter by category to focus on telephony, CRM, healthcare, restaurant, hotel, or KB integrations.
- Each card shows the integration logo and a short summary of what the connection enables.
- Designed as a **capability overview**; configuration remains managed via existing setup flows.

</Accordion>

<Accordion title="Multi-channel dashboard" icon="chart-line" id="multi-channel-dashboard">

Agent Studio’s homepage now better reflects both **voice** and **chat** performance.

![multi-channel-dashboard](/images/release-notes/2511/homepage.png)

**What's new:**
- Split the homepage into **Voice** and **Chat** tabs, with tab visibility based on which channels your project uses.
- Chat tab highlights key KPIs such as engaged conversations, containment rate, total messages, cost savings, and average handle time.
- New charts for chat deployments: engaged conversations over time, messages per conversation, total message volume, top QA reasons, and containment evolution.
- Currency selector for core metrics (USD, GBP, EUR), with automatic conversion.

</Accordion>

<Accordion title="SMS messaging-service IDs" icon="message" id="sms-messaging-service-ids">

![twilio-msg-service](/images/release-notes/2511/twilio.png)

Outbound SMS fields now support Messaging Service IDs.

**What's new:**
- UI accepts Twilio-style Messaging Service IDs like `MGxxxxxxxxxx`.
- Replaces the previous phone-number-only validator, aligning the UI with existing API behaviour.

</Accordion>
