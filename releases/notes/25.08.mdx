The **August 2025** PolyAI Agent Studio release focuses on performance improvements, voice configuration flexibility, new model options, and more powerful call review tools.

Expand the items for details:

<AccordionGroup>

<Accordion title="Home page updates" icon="house">

The Agent Studio homepage now loads faster and presents key metrics more cleanly.

![homepage-optimisation](/images/release-notes/2508/homepage.png)

**What's new:**

- Reduced page load time
- Improved widget layout for better readability
- More consistent alignment across metric cards and charts

</Accordion>

<Accordion title="AI call summaries" icon="file-lines">

Reviewing calls just got faster. Agent Studio now displays an **AI-generated summary** at the top of the Conversation Review screen.

![ai-call-summary](/images/release-notes/2508/ai-summary.png)

**What’s new:**

- New **AI Summary** module in the **Conversation Review** screen
- Generated post‑call using GPT summarization logic

<Note>Appears by default for projects that have signed the **current DPA** and support **Azure AI**. If unavailable, Conversation Details shows by default.</Note>

</Accordion>

<Accordion title="Multi-voice limit increase" icon="waveform">

Projects can now have **up to 10 active voices**, updated from the previous limit of **5** voices.

![multi-voice-limit](/images/release-notes/2508/multi-voice.png)


Voice limits apply per project and can be managed from the settings page.

</Accordion>

<Accordion title="New call review filters" icon="filter">

It is now much easier to filter calls in Conversation Review.

![enhanced-filters](/images/release-notes/2508/table-filters.png)

**What’s new:**

- New filter options for delivery status, channel, and other metadata
- Faster filter response time
- Filter combinations can be applied without resetting the view

<Tip>Use saved filters to make these configurations instantly reusable.</Tip>

</Accordion>

<Accordion title="Updated channel display" icon="list">

Conversation Review now displays additional call attributes and improved channel labels for quicker scanning.

![channel-metadata](/images/release-notes/2508/channel.png)

**What's new:**

- Clearer channel tags for *Inbound*, *Outbound*, and *Agent Chat*
- Additional metadata columns available for custom table views
- Streamlined column naming for consistency across the platform

</Accordion>

<Accordion title="GPT‑5 models in Agent Studio (experimental)" icon="sparkles">

We’ve added **GPT‑5** options to Agent Studio for evaluation and feedback.

![gpt5-models](/images/release-notes/2508/gpt-5.png)

**What's available:**

- **GPT‑5 nano** and **GPT‑5 mini**: Recommended for most Agent Studio use cases (best balance of quality/latency)
- **GPT‑5 (large)**: Included for testing; note it's currently slow at 3–4 seconds or more.
- **GPT‑5 chat (router)**: Routes dynamically across the above models so you can experiment quickly.

<Tip>Switch models from function settings or your project’s model configuration panel.</Tip>

</Accordion>

</AccordionGroup>
