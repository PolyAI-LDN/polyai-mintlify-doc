The **February 2026** PolyAI Agent Studio release focuses on telephony integrations, webchat markdown support, version comparison tools, and variant management improvements.

Expand the items for details:

<AccordionGroup>

<Accordion title="CCaaS telephony integrations" icon="phone" id="ccaas-integrations">

Agent Studio now supports direct integrations with leading CCaaS providers, making it easier to connect your voice agent to existing contact center infrastructure.

**What's new:**
- Native integration support for **Five9**, **Twilio**, and **Zendesk** contact centers.
- Configure telephony connections directly from the Integrations page.
- Streamlined setup flow — after configuring an integration, navigate directly to the **Handoffs** page to complete routing configuration.
- Updated **Handoffs page** reflects telephony integration options for unified call routing management.

</Accordion>

<Accordion title="Markdown in webchat" icon="message" id="webchat-markdown">

Webchat conversations now render markdown formatting, improving readability for complex responses.

**What's new:**
- **Display markdown in webchat** — agent responses with formatting (bold, lists, links, code blocks) render properly in the chat widget.
- **Markdown in conversation review** — webchat channel conversations show formatted markdown when reviewing transcripts.
- Better presentation for structured content like instructions, lists, and code snippets.

</Accordion>

<Accordion title="Compare versions and environments" icon="code-compare" id="version-comparison">

A new comparison tool helps teams understand differences between versions and environments before promoting changes.

**What's new:**
- **Compare versions/environments** side-by-side to review changes before promotion.
- See exactly what changed between Sandbox, Pre-release, and Live environments.
- Updated **Versions page** with improved ordering and navigation.
- Clearer version history and environment state visibility.

</Accordion>

<Accordion title="In-app calling with variants" icon="phone-volume" id="in-app-call-variants">

Test calls now support variant selection, allowing teams to validate variant-specific configurations directly in Agent Studio.

**What's new:**
- Select a specific [variant](/variant-management/introduction) when making in-app test calls.
- Verify variant-specific content, routing, and responses without switching phone numbers.
- **Variant name consistency** — the variant name displayed in the Numbers page now matches the site name for clarity.

</Accordion>

<Accordion title="Delay control improvements" icon="hourglass-half" id="delay-control-improvements">

Delay control responses now display inline with agent responses for a more natural conversation flow.

**What's new:**
- Transition utterances and filler phrases appear **inline** with the agent's final response in conversation review.
- Improved visibility into how delay control affects real conversations.
- Better debugging for [high-latency functions](/function/delay-control).

</Accordion>

<Accordion title="Edit forms for integrations" icon="pen-to-square" id="edit-forms">

Users can now define custom edit forms for integration configurations, improving flexibility for complex setups.

**What's new:**
- Define **custom edit forms** for integration parameters.
- Structured input fields for integration-specific settings.
- Reduces configuration errors with validated form inputs.

</Accordion>

<Accordion title="Voice and channel sync" icon="microphone" id="voice-channel-sync">

Voice configuration now stays in sync across channels and agent settings.

**What's new:**
- **Sync voice channel with agent settings** — changes to voice configuration propagate correctly.
- **Disclaimer voice** automatically updates when the agent voice changes.
- Unified voice configuration across all agent touchpoints.

</Accordion>

<Accordion title="Sample questions expansion" icon="circle-question" id="sample-questions">

Increase the variety of sample questions available for agent testing and onboarding.

**What's new:**
- Expanded pool of sample questions for agent testing.
- More diverse conversation starters for different use cases.

</Accordion>

<Accordion title="Conversation filters update" icon="filter" id="conversation-filters">

The conversation review filters have been refined for better usability.

**What's new:**
- Removed the **Live filter** on conversations — all conversations now appear in a unified view.
- Simplified filtering experience for reviewing call history.

</Accordion>

<Accordion title="Secrets security" icon="lock" id="secrets-security">

The Secrets Vault now hides sensitive information by default for improved security.

**What's new:**
- **API addresses hidden** in the Secrets UI — sensitive endpoints are obscured until explicitly revealed.
- Additional protection against accidental exposure of credentials.

</Accordion>

<Accordion title="Notification banner update" icon="bell" id="notification-banner">

The notification banner has been updated with improved visibility and messaging.

**What's new:**
- Refreshed notification banner design.
- Clearer system alerts and status updates.

</Accordion>

<Accordion title="In-app guidance" icon="graduation-cap" id="pendo-guides">

New in-app guidance helps users discover features and complete common tasks.

**What's new:**
- Additional **Pendo guides** for feature discovery.
- Contextual help for new and existing features.

</Accordion>

</AccordionGroup>
