The **February 2025** PolyAI platform release introduces new tools for knowledge base versioning, function latency control, improved conversation review, and enhanced agent configuration.

Expand the items for details:

<AccordionGroup>

<Accordion title="View knowledge base differences between versions" icon="book-open">
Compare two knowledge base versions side-by-side to track modifications.

![kb-diff](/images/release-notes/kb-diff.png)

**Key benefits:**
- Identify additions, deletions, and changes between knowledge base versions.
- Improve collaboration and oversight by understanding content evolution.
- Reduce manual tracking when updating assistant knowledge.

</Accordion>

<Accordion title="Function latency control" icon="wifi">
Dynamically manage function latency with configurable filler utterances.

![latency-control](/images/release-notes/latency-control.png)

**How it works:**
- Define filler utterances that play if a function takes longer than expected.
- Set delay times and intervals between utterances.
- Improve user experience by avoiding dead air in long-running function calls.

</Accordion>

<Accordion title="Audio management - upload audio" icon="volume-high">
Upload pre-recorded audio directly from the **audio management resource** in Jupiter.

![audio-upload](/images/release-notes/audio-upload.png)

**Key benefits:**
- Easily replace TTS with high-quality voice actor recordings.
- Improve voice consistency across different assistant responses.
- Initial release supports single file uploads, with bulk upload functionality coming soon.

</Accordion>

<Accordion title="Send SMS inline" icon="comment-dots">
The `send_sms` function now executes inline within function calls.

![sms-inline](/images/release-notes/sms-inline.png)

**Key benefits:**
- Prevents agents from confirming SMS delivery before it is sent.
- Enables real-time error handling during function execution.
- Improves accuracy in assistant messaging workflows.

</Accordion>

<Accordion title="Encoding variants in RAG" icon="layer-group">
Reference variant attributes dynamically within knowledge base responses.

![rag-variants](/images/release-notes/rag-variants.png)

**How it works:**
- Add variant attributes by typing `/` or selecting the `+` icon in content fields.
- Reference existing attributes or create new ones dynamically.
- Personalize responses by encoding structured variant data.

</Accordion>

<Accordion title="Managing transition functions in a flow" icon="arrow-right-arrow-left">
Transition functions are now managed centrally within the flow editor.

![transition-functions](/images/release-notes/transition-functions.png)

**Key benefits:**
- Prevents accidental deletions when modifying flows.
- Enables reusing, duplicating, and managing transition functions from a single interface.
- Streamlines assistant configuration and debugging.

</Accordion>

<Accordion title="Conversation review annotations" icon="magnifying-glass">
Users can now annotate conversation turns for **missed topics** or **incorrect ASR transcriptions**.

![review-annotations](/images/release-notes/review-annotations.png)

**How it works:**
- Flag agent turns for **incorrect information**.
- Flag user turns for **missing knowledge base topics** or **wrong ASR transcription**.
- Export annotated calls for quality assurance and training improvements.

</Accordion>

<Accordion title="Agent config management" icon="gears">
Configure agent settings like **opening hours and holiday hours** within the PolyAI platform.

![agent-config](/images/release-notes/agent-config.png)

**Key benefits:**
- Eliminates reliance on external file storage for agent configurations.
- Centralized configuration management within the UI.
- Simplifies maintaining business logic for multiple agents.

</Accordion>

<Accordion title="Enterprise overview dashboard" icon="chart-bar">
A new **Enterprise Overview Dashboard** provides key operational insights.

![enterprise-dashboard](/images/release-notes/enterprise-dashboard.png)

**Key benefits:**
- Monitor total calls, call duration, words generated, and key metrics.
- Improve oversight of agent performance and containment rates.
- Works alongside custom dashboards for deeper analytics.

</Accordion>

</AccordionGroup>
