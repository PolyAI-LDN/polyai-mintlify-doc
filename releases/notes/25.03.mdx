The **March 2025** PolyAI Agent Studio release includes enhancements to dashboards, voice management, knowledge base performance, and real-time collaboration features.

Expand the items for details:

<AccordionGroup>

<Accordion title="Overview dashboard enhancements" icon="chart-pie">
A redesigned overview dashboard now includes additional metrics and improved performance.

![enhanced-overview-dashboard](/images/release-notes/2503/new-dashboard.png)

**What's new:**
- Containment rate, total calls, SMS stats, and function call metrics.
- Hourly refresh for near real-time data.
- Improved performance and usability with a dark header for contrast.

</Accordion>

<Accordion title="Safety dashboard enhancements" icon="shield-check">
The new safety dashboard mirrors the updated overview dashboard experience.

![enhanced-safety-dashboard](/images/release-notes/2503/new-safety.png)

**What's improved:**
- Faster loading and improved performance.
- Hourly data refresh.
- Unified styling with a dark header for visual clarity.

</Accordion>

<Accordion title="Open matched topics in a new tab" icon="arrow-up-right-from-square">
You can now click matched [knowledge base](/knowledge-base/introduction) topics in [conversation review](/analytics/conversations/review) to open them in a new browser tab.

This enables deeper inspection of topics without losing your place in the review flow.

![matched-topic-new-tab](/images/release-notes/2503/topic-tab.png)

</Accordion>

<Accordion title="Live collaboration and edit history" icon="users">
Agent Studio now shows who is editing a draft in real time and tracks edit history.

**How it works:**
- Get notified when another builder saves changes.
- Copy your unsaved edits before refreshing to avoid losing work.
- View full edit logs using the **history** panel.

</Accordion>

<Accordion title="Version info added to conversation review" icon="clock-rotate-left">
Each reviewed conversation now displays the **agent version** it ran on.

![review-version](/images/release-notes/2503/review-version.png)

**Why it matters:**
- Helps identify environment-specific behaviour.
- Improves accuracy when debugging or performing quality reviews.

</Accordion>

<Accordion title="Add new voices in Agent Studio" icon="microphone">
You can now add new voices directly within Agent Studio.

![add-voice](/images/release-notes/2503/add-voice.png)

<Tip>Refer to your voice provider's documentation to locate the correct voice ID for upload.</Tip>

</Accordion>

<Accordion title="Filter voices by language, accent, and gender" icon="filter">
The voice selection page now includes filtering options for easier browsing.

![voice-filters](/images/release-notes/2503/voices-redesign.png)

**Available filters:**
- Language
- Accent
- Gender

</Accordion>

<Accordion title="Faster knowledge base imports" icon="rocket">
CSV uploads to the knowledge base are now powered by GPU-backed embedding inference.

**What's improved:**
- Faster processing for large knowledge base files.
- Reduced wait times when importing content.

</Accordion>

<Accordion title="Send SMS during a function call" icon="message">
SMS messages are now sent inline during [function](/function/introduction) execution, rather than being queued post-execution.

**Why it matters:**
- Enables immediate error handling for SMS failures.
- Prevents agents from falsely confirming SMS delivery.
- Improves transparency and reliability during runtime.

</Accordion>

<Accordion title="Improved KB search for actions" icon="magnifying-glass">
The knowledge base search bar now supports action keywords such as [SMS](/sms/introduction), [handoff](/call-handoff/introduction), function, and [variant](/variant-management/introduction).

**What's new:**
- These keywords are now fully indexed and searchable.
- Previously, users had to rely on **Ctrl+F / Cmd+F** to find them.

</Accordion>

</AccordionGroup>
