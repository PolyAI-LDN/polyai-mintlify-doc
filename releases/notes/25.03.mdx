The **March 2025** PolyAI Agent Studio release includes enhancements to dashboards, voice management, knowledge base performance, and real-time collaboration features.

Expand the items for details:

<AccordionGroup>

<Accordion title="Send SMS during a function call" icon="message">
SMS messages are now sent **inline during the function call**, rather than being queued after execution.

![send-sms-inline](/images/release-notes/2503/send-sms-inline.png)

**Why it matters:**
- Reduces the risk of falsely confirming SMS delivery.
- Enables immediate error handling if SMS sending fails.
- Provides a more reliable and transparent user experience.

</Accordion>

<Accordion title="Open matched topics in a new tab" icon="arrow-up-right-from-square">
You can now click matched topics in **conversation review** and open them in a new browser tab.

![matched-topic-new-tab](/images/release-notes/2503/matched-topic-tab.png)

**Key benefits:**
- Keeps your place in the review panel.
- Allows deeper inspection of KB topics without breaking workflow.

</Accordion>

<Accordion title="Live collaboration and edit history" icon="users">
Agent Studio now shows who's working on a draft in real time, and tracks edit history.

![real-time-collaborators](/images/release-notes/2503/live-collab.png)

**How it works:**
- Get alerts when another builder saves changes.
- Copy your unsaved edits before refreshing.
- Review edit logs using the **history** panel.

</Accordion>

<Accordion title="Improved KB search for actions" icon="magnifying-glass">
The knowledge base search bar now supports action terms like **SMS**, **handoff**, **function**, and **variant**.

![kb-search-actions](/images/release-notes/2503/kb-search.png)

**Previously:** These tags were only searchable via **Cmd + F**.

**Now:** They're fully indexed in the KB search interface, making them faster to locate and reference.

</Accordion>

<Accordion title="Import and export variants" icon="file-arrow-up">
You can now upload or download variant data as CSV from the **variant management page**.

![variant-csv](/images/release-notes/2503/variant-csv.png)

**Key benefits:**
- Easily back up or transfer variant data between projects.
- Requires **Edit** access to use the import function.

</Accordion>

<Accordion title="Use the same transition function across flows" icon="shuffle">
Transition functions are now scoped to individual flows.

![transition-flow-scope](/images/release-notes/2503/transition-flow.png)

**Why it matters:**
- Reuse the same transition function name across different flows.
- Avoid naming conflicts or global overrides.

</Accordion>

<Accordion title="Faster knowledge base imports" icon="rocket">
CSV uploads to the knowledge base are now powered by **GPU-backed embedding inference**.

![embedding-speedup](/images/release-notes/2503/embedding-speed.png)

**What's improved:**
- Significant speed increase for larger KBs.
- Reduced processing time after import.

</Accordion>

<Accordion title="Version info added to conversation review" icon="clock-rotate-left">
Each conversation now displays the **agent version** it ran on.

![review-version](/images/release-notes/2503/review-version.png)

**Use cases:**
- Debugging differences across environments.
- Aligning team reviews with versioned releases.

</Accordion>

<Accordion title="Add new voices in Agent Studio" icon="microphone">
You can now add and manage assistant voices directly in Agent Studio.

![add-voice](/images/release-notes/2503/add-voice.png)

**Key benefits:**
- Simplifies voice updates without external setup.
- Improves voice management within the platform.

</Accordion>

<Accordion title="Filter voices by language, accent, and gender" icon="filter">
The voice selection page now includes filters for easier navigation.

![voice-filters](/images/release-notes/2503/voice-filters.png)

**Available filters:**
- **Language**
- **Accent**
- **Gender**

</Accordion>

<Accordion title="Enhanced overview dashboard" icon="chart-pie">
A redesigned overview dashboard now includes additional metrics and improved performance.

![enhanced-overview-dashboard](/images/release-notes/2503/overview.png)

**What's new:**
- Containment rate, call minutes, SMS stats, and function metrics.
- Loads faster with hourly refreshes.
- Includes a dark header for contrast and usability.

</Accordion>

<Accordion title="Enhanced safety dashboard" icon="shield-check">
A new safety dashboard now mirrors the improved overview dashboard experience.

![enhanced-safety-dashboard](/images/release-notes/2503/safety.png)

**Key benefits:**
- Shared design and refresh cadence with the overview dashboard.
- Faster loading and better layout clarity.

</Accordion>

</AccordionGroup>
