    <Update label="March 2025">
      <Card>![march-release-main-image](/images/release-notes/2503/main.png)</Card>

      Click the **bolded** titles for a detailed breakdown of the released feature, or navigate to the [notes tab](./notes/25.03) on the sidebar for the full release notes.

      - **[Send SMS immediately during function calls](./notes/25.03#send-sms-inline)**: SMS messages now send in real time as part of the function, so agents won’t falsely confirm delivery if the message fails.
      - **[Open matched topics in a new tab](./notes/25.03#matched-topic-new-tab)**: In conversation review, clicking a matched topic opens it in a new browser tab—letting you keep your place while checking the KB.
      - **[Live collaboration support](./notes/25.03#real-time-collaborators)**: Agent Studio now shows who else is editing a draft, alerts you when updates are saved, and lets you review edit history to avoid accidental overwrites.
      - **[Improved KB search for actions and tags](./notes/25.03#search-by-action)**: You can now search for SMS, handoff, and function names in the knowledge base using the main search bar—not just Cmd+F.
      - **[Import and export variants as CSVs](./notes/25.03#variant-import-export)**: From the variant management page, export your variants or upload new ones via CSV. Edit access required for imports.
      - **[Flow-specific transition functions](./notes/25.03#unique-transition-functions)**: You can now reuse the same transition function name across different flows without conflict.
      - **[Faster knowledge base imports](./notes/25.03#kb-embedding-speedup)**: CSV uploads and embedding updates are now backed by GPU-powered inference for faster performance.
      - **[Version tracking in conversation review](./notes/25.03#review-version-info)**: Each conversation now shows which version of the agent it ran on—useful for debugging or reviewing changes over time.
      - **[Add voices directly in Agent Studio](./notes/25.03#add-voice-agent-studio)**: Voices can now be added or managed without leaving the platform.
      - **[Filter voices by language, accent, or gender](./notes/25.03#voice-page-filters)**: The voices page now supports helpful filters to speed up selection.
      - **[New overview dashboard](./notes/25.03#enhanced-overview-dashboard)**: A redesigned dashboard showing call volumes, durations, containment rate, SMS and function stats—refreshed hourly.
      - **[New safety dashboard](./notes/25.03#enhanced-safety-dashboard)**: A refreshed version of the safety dashboard to match the new overview, with faster load times and a clearer layout.
    </Update>
