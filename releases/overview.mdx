PolyAI releases monthly platform updates, which are versioned using the following syntax:

| Segment    | Meaning                                   | Example               |
|------------|-------------------------------------------|-----------------------|
| **24**     | The calendar year of the release. | *2024*                |
| **4**      | The quarter. | *Q4 (October–December)* |
| **1**      | The specific monthly release within that quarter. | *First update of Q4*  |

For example, version 24.4.1 is the first release of the fourth quarter in 2024, or the [October 2024](#24-4-1) release.
