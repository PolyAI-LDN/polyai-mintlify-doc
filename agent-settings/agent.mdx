Use the Agent page to define how your agent introduces itself and communicates. The greeting is the first thing a caller hears — if it sounds wrong, callers lose confidence before the conversation begins.

![agent](/images/agent.png)

Configure your agent's greeting, personality, and role in **Build > Agent > Agent**.

## Greeting

The agent's opening line. You can include [function calls](/tools/introduction) and [variant calls](/variant-management/introduction). The greeting goes directly to [TTS](https://en.wikipedia.org/wiki/Speech_synthesis) without LLM processing (see [conversation flow](/essentials/order)).

## Personality

Sets the tone and communication style (friendly, professional, empathetic, casual). Define a custom personality to match your brand.

## Roles

Specifies the agent's function (customer service, sales, technical support). Define role behavior further in [Rules](./rules).
