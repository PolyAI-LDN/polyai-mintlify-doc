The **Model** section in the Agent Settings allows you to select the Large Language Model (LLM) that best suits your agent's requirements.

Below is an overview of the models currently available:

## [GPT-4o Mini](https://platform.openai.com/docs/models#gpt-4o)
The lightweight version of GPT-4o, offering faster performance and cost-efficiency. Best suited for simple queries and applications prioritizing speed and affordability.

## [GPT-4o](https://platform.openai.com/docs/models#gpt-4o)
An optimized version of GPT-4, providing high-quality responses with reduced latency. Ideal for scenarios requiring both accuracy and responsiveness.

## [GPT-4](https://platform.openai.com/docs/models#gpt-4)
The most advanced model for generating high-quality, detailed responses. Recommended for complex tasks requiring precision and contextual understanding.

## [GPT-3.5 (0125)](https://platform.openai.com/docs/models#gpt-3-5-turbo)
An enhanced version of GPT-3.5 with optimizations for stability and specific use cases. Offers a balance between performance and cost-effectiveness.

## GPT-3.5
A reliable and cost-efficient option for applications where speed and affordability are the primary considerations. Suitable for straightforward interactions and real-time responses.

## Configuring the model

  ![llm-use](/images/model.png)

To configure the model for your agent:
1. Navigate to the **Large Language Model** section in Agent Settings.
2. Select the desired model from the dropdown list.
3. Save your settings to apply the changes.

For more detailed information on each model, refer to the [OpenAI Model Documentation](https://platform.openai.com/docs/models).
