Configure project-wide settings that apply across all channels and environments. Misconfigured safety filters can allow harmful content through to callers or block legitimate responses — review these settings before going live.

Access these settings from **Configure > General** in the sidebar.

## Safety filter defaults

Set default content safety filters for your agent. These filters control what content is allowed in agent responses.

<Note>
Channel-specific overrides can be configured in [Voice configuration](/voice/voice-configuration) and [Chat configuration](/webchat/chat-configuration).
</Note>

| Category | Description |
|----------|-------------|
| **Violence** | Controls filtering of violent content |
| **Hate** | Controls filtering of hateful or discriminatory content |
| **Sexual content** | Controls filtering of sexually explicit content |
| **Self-harm** | Controls filtering of self-harm related content |

Each category has two controls:
- **Enabled/disabled toggle** — turn filtering on or off for the category
- **Strictness level** — set to Loose, Medium, or Strict to control how aggressively content is filtered

For more details on how safety filters work, see the [Safety dashboard](/analytics/dashboards/safety).
