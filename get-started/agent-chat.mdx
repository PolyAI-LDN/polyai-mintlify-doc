Test your agent through in-browser calling, webchat, or phone calls.

## In-browser calling

<img width="425" src="/images/get-started/test-call.png" />

In-browser calling is the fastest way to test your agent's voice and automatic speech recognition (ASR) without setting up telephony services. It uses your device's microphone directly.

### How to start an in-browser call

1. **Open Agent Studio**:
  Log in at [Agent Studio](https://studio.us.poly.ai/) and select your agent from the dashboard.

2. **Click the phone icon**:
  In the top-right corner of the studio page, click the phone icon <Icon icon="phone" size={24} />.

3. **Select an agent version**:
  Use the dropdown menu to select the version of your agent to test.

4. **Begin testing**:
  Interact with your agent in real-time directly from your browser.

## Webchat

Interact with your agent using text-based chat directly from Agent Studio, without needing a phone number or external service.

### How to start a webchat

1. **Go to the agent main page**:
   Log in to your account and go to the agent main page.

   ![agent](/images/get-started/agent.png)

2. **Click the webchat icon**:

   ![start-a-webchat](/images/get-started/start-a-webchat.png)

### Webchat interface

- **Chat window**: The main area where you can type and receive messages.
- **Tool calls**: Tools (including functions) that the agent can execute based on your commands.
- **Topic citations**: Topics that the agent can recognize and respond to appropriately.
- Click the Settings icon to show or hide the tool calls and topic citations settings.

![webchat-interface](/images/get-started/webchat.png)

## Phone calls

Connect with your agent by phone to test its voice and ASR in a production-like environment. This method involves setting up a phone number, often through a service like [Twilio](/telephony/introduction).

### How to start a phone call

1. **Go to the agent main page**:
   Log in to your account and go to the agent main page.

   ![agent](/images/get-started/agent.png)

2. **Buy or connect a number**:
   Click **"Add number"** to purchase a phone number or connect an existing Twilio account. For detailed instructions, visit the [Numbers](/telephony/introduction) page (**Configure > Numbers**).

   ![add-number](/images/add-number.png)

3. **Configure the number**:
   Follow the instructions to configure your phone number with the agent.

   ![telephony](/images/quickstart/telephony.png)

## Language selection

For [multilingual agents](/agent-settings/multilingual), a **language dropdown** appears in the Agent Chat panel. Use it to select which language to test with — similar to how you select variants. The dropdown defaults to the main language on the first turn.

## Tips for testing

- **Clear communication**: Speak clearly to ensure the agent understands your input.
- **Use keywords**: Use specific keywords and phrases to trigger the agent's tools.
- **Test variety**: Test with different speakers, accents, and scenarios to evaluate your agent's speech recognition accuracy.
- **Knowledge testing**: Test items added to your [Managed Topics](/managed-topics/introduction) or [global rules](/agent-settings/rules), including [TTS pronunciations](/agent-settings/rules#pronunciations).
- **Language testing**: For multilingual agents, switch languages mid-conversation to test detection and voice switching.
