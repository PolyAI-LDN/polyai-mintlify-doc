Agent Studio is organized into five main navigation sections, each accessible from the left sidebar. This page walks you through what each section contains and when you'd use it.

<Note>
  Your sidebar may look different depending on your [role and permissions](/user-management/access-control-scope). If a section is missing, ask your admin to check your access level.
</Note>

## Interface layout

Agent Studio uses a three-panel layout:

- **Left sidebar** — Navigate between the five main sections (Analytics, Build, Channels, Configure, Deployments) and their sub-sections
- **Center canvas** — The main working area where you view and edit content
- **Right sidebar** — Contextual details, previews, or settings related to what you've selected

## Navigation sections

### Analytics

{/* TODO: Add screenshot of the Analytics section in the sidebar */}

Review conversation data and track agent performance.

| Sub-section | What it does |
|---|---|
| **Smart analyst** | Ask questions about your agent's performance in natural language |
| **Conversations** | Browse call transcripts, listen to recordings, and review individual interactions |
| **Agent analysis** | View performance metrics, trends, and aggregate insights |

<Card title="Analytics docs" icon="chart-bar" href="/analytics/conversations/introduction">
  Learn more about reviewing conversations and tracking performance.
</Card>

### Build

{/* TODO: Add screenshot of the Build section in the sidebar */}

Create and configure your agent's behavior, knowledge, and conversation logic.

| Sub-section | What it does |
|---|---|
| **Agent** | Set your agent's personality, greeting, model, and behavioral rules |
| **Knowledge** | Add managed topics and connect external data sources |
| **Flows** | Design multi-step conversation workflows |
| **Tools** | Write Python functions and manage integrations |
| **Variant management** | Configure multi-site or multi-brand variations |
| **Call handoffs** | Set up transfer destinations and routing rules |
| **SMS** | Configure text messaging actions |
| **Configuration builder** | Build and manage configuration schemas |
| **Test suite** | Run automated tests against your agent |

<Card title="Build docs" icon="hammer" href="/agent-settings/introduction">
  Start with agent settings, then explore knowledge, flows, and tools.
</Card>

### Channels

{/* TODO: Add screenshot of the Channels section in the sidebar */}

Configure how your agent sounds and how users interact with it.

| Sub-section | What it does |
|---|---|
| **Voice** | Choose and customize your agent's voice, including TTS settings |
| **Response control** | Manage stop keywords and pronunciation overrides |
| **Audio management** | Configure hold music, IVR prompts, and other audio |
| **Speech recognition** | Tune how your agent interprets spoken input |
| **Voice configuration** | Advanced voice parameter settings |
| **Chat** | Set up the web chat widget and chat-specific behavior |

<Card title="Channels docs" icon="satellite-dish" href="/voice/introduction">
  Start with voice setup, then explore chat and advanced audio settings.
</Card>

### Configure

{/* TODO: Add screenshot of the Configure section in the sidebar */}

Manage project settings, phone numbers, integrations, and dashboards.

| Sub-section | What it does |
|---|---|
| **General** | Project-wide settings and preferences |
| **Numbers** | Buy, manage, and configure phone numbers |
| **APIs** | Manage API keys and access |
| **Integrations** | Connect third-party services (CRM, telephony, etc.) |
| **Metrics** | Define custom metrics for tracking |
| **Dashboards** | View standard, safety, and custom performance dashboards |
| **CSAT** | Configure customer satisfaction surveys |
| **User management** | Invite users and manage roles and permissions |
| **Call data** | Access transcripts, recordings, and data exports |

<Card title="Configure docs" icon="gear" href="/settings/introduction">
  Start with general settings, then set up numbers, integrations, and dashboards.
</Card>

### Deployments

{/* TODO: Add screenshot of the Deployments section in the sidebar */}

Promote your agent through environments and track version history.

| Sub-section | What it does |
|---|---|
| **Environments** | Move your agent from Sandbox to Pre-release to Live |
| **Project history** | View past versions, compare changes, and roll back if needed |

<Card title="Deployments docs" icon="rocket" href="/environments-and-versions/introduction">
