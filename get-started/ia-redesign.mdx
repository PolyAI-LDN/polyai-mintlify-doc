On **24 June 2026** the Agent Studio sidebar was reorganized. The new navigation is on by default for every user — a temporary **revert toggle** is available in your profile menu while you adjust.

This page walks through the new layout, what moved, and where to find the features you use most.

## Why we changed it

As Agent Studio has grown, the sidebar accumulated ~28 pages spread across loosely related groups (**Build**, **Channels**, **Configure**). Common tasks – tuning behavior, adding knowledge, configuring a channel – often required jumping across unrelated sections.

The new IA groups features by **what you are doing**, not by which team shipped them. Related settings now live together, and the top-level page count drops from ~28 to 16.

## The new top-level sections

| Section | What it contains |
|---------|------------------|
| **Home** | Studio Assistant, our natural-language builder. Describe what you want to do in plain language and it does the work for you. |
| **Analytics** | View your KPIs and manage custom metrics. |
| **Conversations** | Browse the full history of your conversations, with PolyScores, search and filtering. |
| **Custom Dashboards** | Build, manage and view custom dashboards. |
| **Knowledge** | Manage your agent's knowledge base, including FAQs, external sources and variants. |
| **Flows** | Create and manage flows for structured, step-by-step user journeys (e.g. ID&V). |
| **Tools** | Create and manage the tools your agent can use. |
| **Testing** | Create, manage and run simulation tests across all channels. |
| **Voice** | Manage all your voice settings. |
| **Messaging** | Manage all your messaging channel settings (e.g. webchat, SMS, email etc.). |
| **Integrations** | Set up and manage your integrations across app integrations, APIs and MCP connections. |
| **Deployments** | Manage deployments, view deployment history and run A/B tests. |
| **Widgets** | Create, manage and deploy your widgets across channels and surfaces. |
| **Account** | Manage your account settings. |

The previous **Build / Channels / Configure** top-level groups are gone — every page that lived under them now lives in one of the sections above.

## The top toolbar

The toolbar at the top of Agent Studio has been redesigned. The full reference is in [Top toolbar](/get-started/top-toolbar); here's the summary of what changed:

| Control | Status | Notes |
|---------|--------|-------|
| **Agent name** | Unchanged | Still on the left. Click to rename. |
| **Project selector** | New | Switch between projects without going back to the workspace. |
| **Branch selector** | Unchanged | Switch between Draft, Sandbox, Pre-release, Live, and feature branches. |
| **Chat with Studio Assistant** | New | Opens [Studio Assistant](/studio-assistant/introduction) from any page. |
| **Share** | New | Generate a preview link to share the current branch. |
| **Preview agent** | Unchanged | Live test conversation with the current branch. |
| **Promote** | Reworked | Single button consolidating release & branch history, **Compare with Live**, **Move to Staging**, **Publish to Live**, **Run A/B test**, and **Discard changes**. |
| **Global search** | New | Press <kbd>⌘ K</kbd> / <kbd>Ctrl K</kbd> to search across pages, settings, and knowledge from any screen. |
| **Add/view numbers** | Removed | Manage numbers under [Voice > Numbers](/voice-channel/numbers/introduction). |

## What's been removed

- **Agent Analysis** has been sunset. Per-call diagnosis lives at [Analytics > Conversations > Diagnosis](/analytics/conversations/diagnosis); aggregate trends live in [Dashboards](/analytics/dashboards/introduction) and [Smart Analyst](/smart-analyst/introduction).
- **Project history** (the standalone History page) has been sunset. Version history, event types, authors, timestamps, and side-by-side diffs all live directly on the [Deployments](/environments-and-versions/introduction) page. See [Compare versions](/environments-and-versions/diffs).
- **Luzmo dashboards** have been sunset. Use [Standard dashboards](/analytics/dashboards/standard), [Safety dashboards](/analytics/dashboards/safety), or [Custom dashboards](/analytics/dashboards/custom).
- **Add/view numbers** has been removed from the top toolbar — number management lives at [Voice > Numbers](/voice-channel/numbers/introduction).

## Old URLs still work

Every page in the old sidebar redirects to its new location, so existing bookmarks, links in old emails, and links in your help-desk macros continue to work. You don't need to update anything urgently.

## Reverting (temporary)

If you need the old sidebar while you migrate internal docs or run training:

1. Open your profile menu (top-right).
2. Toggle **Use previous navigation**.

The revert toggle will stay available for a short period after launch, then be removed.
