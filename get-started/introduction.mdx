Agent Studio gives you everything you need to build, test, and deploy conversational AI agents through a visual interface. No programming experience required.

<Note>
  If you are a developer looking to extend your agent with Python functions and API integrations, see [Extend with code](/extend/introduction) instead.
</Note>

## What you can do

<CardGroup cols={3}>
  <Card title="Create agents" icon="plus">
    Set up new agents with a name, language, voice, and greeting in minutes
  </Card>
  <Card title="Add knowledge" icon="brain">
    Build your agent's knowledge base with topics, FAQs, and connected data sources
  </Card>
  <Card title="Design conversations" icon="diagram-project">
    Create multi-step flows for bookings, forms, and complex interactions
  </Card>
  <Card title="Configure voice" icon="microphone">
    Choose from a library of voices and fine-tune speech settings
  </Card>
  <Card title="Deploy and monitor" icon="chart-line">
    Promote agents through environments and track performance with dashboards
  </Card>
  <Card title="Connect integrations" icon="plug">
    Link to telephony, CRM, and third-party services without writing code
  </Card>
</CardGroup>

## Get started

<Steps>
  <Step title="Build your first agent">
    Follow the [quickstart guide](/get-started/quickstart) to create an agent, add knowledge, test it, and deploy — all in under 10 minutes. This gets you a working prototype. For a production-ready agent with integrations, custom functions, and tuned knowledge, expect days to weeks depending on complexity.
  </Step>
  <Step title="Learn the platform">
    Work through [PolyAcademy](/learn/guides/introduction) for structured, hands-on training that covers everything from basic setup to advanced configuration.
  </Step>
  <Step title="Go deeper">
    Explore the sections below to configure every aspect of your agent.
  </Step>
</Steps>
