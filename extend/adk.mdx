The Agent Development Kit (ADK) is a Python CLI and package for building PolyAI agents locally. It gives you tools for conversation flow control, tool integration, multi-agent management, and deployment — all from your local environment.

Pull your project as YAML and Python files, edit with any tool, and push changes back to Agent Studio.

<Note>
ADK is currently in the **Early Access Program**. [Sign up for Early Access](https://fehky.share-eu1.hsforms.com/2oSGLpUctRvyqXcb6K44DAQ) to get started.
</Note>

## How it works

ADK uses a pull-edit-push workflow:

<Steps>
  <Step title="Pull">
    Pull your Agent Studio project to your local machine as YAML configuration and Python function files.
  </Step>
  <Step title="Edit">
    Edit with any tool – IDE, scripts, or AI coding assistants. YAML for configuration, Python for functions.
  </Step>
  <Step title="Push">
    Push changes back to Agent Studio. Your agent updates immediately in the target environment.
  </Step>
</Steps>

ADK is fully compatible with Agent Studio and the APIs. Switch between surfaces at any time.

## Who it's for

- **Development teams** – version control, CI/CD, and code review for agent development
- **Teams at scale** – manage many agents from a single environment
- **Automation** – generate and update agents programmatically
