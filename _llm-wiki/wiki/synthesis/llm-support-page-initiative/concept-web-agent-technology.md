# Web agent technology

PolyAI's existing web-agent product. The thing this initiative is figuring out how to deploy well.

## What it is at a high level

Conversational AI agents that operate on web surfaces — browsers, chat interfaces, embedded widgets. Beyond that, the Slack source doesn't go into detail; it treats the existence and capability of the tech as common knowledge inside PolyAI.

## How it relates to the support-page concept

The support-page redesign is one specific deployment pattern for the web agent. Most current deployments (per [Christopher Osborne](people-christopher-osborne.md)) follow the "webchat widget bolted on" pattern; this initiative argues for going further — the agent _is_ the support page.
