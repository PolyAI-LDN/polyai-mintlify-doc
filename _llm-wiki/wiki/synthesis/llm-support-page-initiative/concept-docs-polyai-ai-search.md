# docs.poly.ai AI-assisted search

The LLM-powered search experience built into PolyAI's public documentation site. Importantly: **embeddable in any front-end dev environment.**

## Why it matters for this initiative

[Aaron Forinton](people-aaron-forinton.md) flagged it explicitly: a front-end dev can drop the AI-assisted search into another site and give users an LLM experience over PolyAI's docs immediately. That's the fastest possible demonstration of "chat-first support" using only assets that already exist.

## Use as a prototype shortcut

For demo / pitch purposes — including the [Quicken pitch](quicken-pitch-plan.md) — embedding the existing search into a support-page mock-up is a credible way to show the experience without building new infrastructure. It's not the final product (which would be tuned to a customer's own knowledge base), but it proves the pattern.
