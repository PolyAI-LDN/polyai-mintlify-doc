For customers who want to manage agents programmatically rather than (or in addition to) via Studio.

## ADK

The [Agent Development Kit](https://docs.poly.ai/extend/adk) is PolyAI's library for managing agents from code — useful for automated build/test/deploy pipelines, multi-agent fleet management, and CI-style gating around the [test suite](../concepts/test-suite.md).

## Configuration builder

[Configuration builder](https://docs.poly.ai/configuration-builder/introduction) — the structured-config approach to building agents (vs the visual Studio UI).

## Code-driven flows

Beyond no-code [Flows](../entities/flow.md), the Advanced tab covers Python-driven flows:

- [Transition functions](https://docs.poly.ai/flows/transition-functions)
- [Flow object](https://docs.poly.ai/flows/object)
- [ASR biasing in flows](https://docs.poly.ai/flows/asr-biasing)
- [DTMF](https://docs.poly.ai/flows/dtmf)
- [Few-shot prompting](https://docs.poly.ai/flows/few-shot-prompting)
