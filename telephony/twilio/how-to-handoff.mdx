title: 'Handoff integration'
description: 'Set up handoff destinations with Twilio'
---

Configure handoff destinations to transfer calls to a specified number using Twilio.

## Prerequisites

The agent must be integrated with a Twilio phone number.

## Setup

1. Click **Call Handoffs** in the sidebar
2. Click **Add Handoff**
3. Fill in the details:
   - **Handoff destination name** - A descriptive name
   - **Extension/Number** - The transfer number (must start with `+`)
   - **Description** - Purpose of this handoff
4. Click **Add**
