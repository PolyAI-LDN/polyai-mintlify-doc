The [Model Context Protocol](https://modelcontextprotocol.io/) (MCP) is an open standard for connecting AI systems to tools. PolyAI works with MCP in **two directions**, and which one you want depends on what you're trying to do.

<CardGroup cols={2}>
  <Card title="PolyAI MCP server" icon="server" href="/mcp/server/introduction">
    **Connect a client in.** Point Claude Code, Cursor, or your own agent at PolyAI's hosted MCP server to build, manage, and monitor your agents conversationally — no direct API calls to write.
  </Card>
  <Card title="Connect external servers" icon="plug" href="/mcp/connect-external-servers">
    **Connect your agent out.** Add a third-party MCP server in Agent Studio so your PolyAI agent can call its tools during live conversations.
  </Card>
</CardGroup>

## Which direction do I want?

| You want to… | Use | Direction |
| --- | --- | --- |
| Manage PolyAI agents from an AI coding tool or custom agent | [PolyAI MCP server](/mcp/server/introduction) | Client → PolyAI |
| Let your live agent look up data or trigger actions in another system | [Connect external servers](/mcp/connect-external-servers) | PolyAI → server |

<Note>
The two are independent. You can connect a client to PolyAI's MCP server and separately wire external MCP servers into your agent — they don't interact.
</Note>
