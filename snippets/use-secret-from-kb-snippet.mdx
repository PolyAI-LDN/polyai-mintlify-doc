
The `conv.utils.get_secret` method lets you securely retrieve the contents of secrets added to your account.

## Usage

Use `conv.utils.get_secret()` inside any function that has access to the `conv` object:

```python
secret_value = conv.utils.get_secret('name of your secret')
```

### Returns
#### For a key-value type secret
For a key-value type secret, the function returns a dictionary of key-value pairs. Example:

```python
{
    "secret_key": "secret_value"
}
```
You can access this like a standard Python dictionary in your function code.


#### For a single type secret

For a single type secret, the function returns the string containing the `secret_value`. Example:

```python
{
    "secret_key": "secret_value"
}
```

### Tips

- **Check Available Secrets**: There is a box called "Secrets" on the top right corner of the "Function Definition" box in the Function Editor. Click that to see what secrets your agent has access to.
- **Permission Issues**: If you cannot see the secret in the "Secrets" box, please refer to the [Access Control for Secrets](/secrets/how-to-access-control) page to add permission for your function.
- **Copy Code Snippet**: In the "Secrets" box, you can also copy the code snippet for accessing the secret directly into your function code.
