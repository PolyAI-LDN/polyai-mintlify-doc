<Info>
**This section is for external developers and partners** building applications on top of the PolyAI platform using APIs and SDKs. If you are configuring agents inside Agent Studio, visit the [Help Center](/home) instead.
</Info>

The Agent Development Kit (ADK) provides programmatic access to PolyAI's conversational AI capabilities for teams building custom integrations, embedding agents in their own products, or automating agent management at scale.

## What's coming

This section is under active development. It will include:

- **SDKs** — Client libraries for common languages
- **API guides** — Step-by-step integration walkthroughs
- **Agent management** — Programmatic agent creation and configuration
- **Webhooks and events** — Real-time event subscriptions
- **Authentication** — OAuth and API key management for external access
