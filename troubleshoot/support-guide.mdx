If you need help using PolyAI Agent Studio or have questions about your deployment, we’re here to support you.

## Getting help

For common issues, feature guides, and how-tos, check out the rest of the documentation site using the sidebar or search bar above.

If you can’t find what you’re looking for:

<Callout title="Contact Support" type="info">
Email us at [support@polyai.com](mailto:support@polyai.com) with as much detail as possible, including:
- A link to the relevant project (if applicable)
- A brief description of the issue
- Steps to reproduce, if possible
</Callout>

## Reporting bugs

If you believe you’ve found a bug in the product:

1. Reproduce the issue and capture any relevant screenshots or logs.
2. Note the browser and environment (e.g. Chrome, Agent Studio, staging or live).
3. Email [support@polyai.com](mailto:support@polyai.com) or post in your internal Slack support channel.
