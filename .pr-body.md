Initial documentation for Studio Assistant – the AI copilot built into PolyAI Studio.

Sourced from `polyai-ldn/platform_ui` (`apps/agentic-builder/src/prompts/system.ts` and the Glot Assistant UI) so the descriptions match what the assistant actually does.

## What's in the page
- **Overview** of what Studio Assistant is and where it lives in Studio.
- **Modes** – Agent / Plan / Ask – explained with a "when to use" table, mirroring the in-product mode toggle.
- **How it works** – the read-context → plan → execute → explain loop.
- **Working effectively** – prompt-quality guidance (project context, Plan mode for non-trivial work, Ask mode for exploration, iterative building).
- **Example prompts** grouped by Building / Updating / Exploring / Planning.
- **Limits** – drafts only, never auto-publishes, business knowledge comes from the user.
- **Related pages** – Flows, Managed topics, Environments and versions.
